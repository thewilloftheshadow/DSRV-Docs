PlaceholderAPI Placeholders

These placeholders are only available with PlaceholderAPI installed. They can be accessed by other plugins that have PlaceholderAPI placeholder compatibility.

The placeholders are integrated into DiscordSRV; you do not need an expansion for them to work.

Some placeholders require EnablePresenceInformation to be enabled!

These placeholders are:

  • %discordsrv_guild_owner_game_name%
  • %discordsrv_guild_owner_game_url%
  • %discordsrv_guild_members_online%
  • %discordsrv_linked_online%
  • %discordsrv_user_online_status%
  • %discordsrv_user_game_name%
  • %discordsrv_user_game_url%

Please note that to use these placeholders you need to have the Presence Intent enabled on your bot using the Discord developer portal. You enable this intent the same way you enabled the Server Members Intent.
